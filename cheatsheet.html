HTML

HTML stands for Hyper Text Markup Language. HTML provides structure to any website. HTML describe elements to browswer on render any page. Any HTML docuement always start with "DOCTYPE HTML" declartion.

Tag Name Description
<table> Table The wrapper elements for all HTML tables
<h1 to h6> Heading The wrapper elements for all HTML headings. It can be defined with <h1 > to <h6> tags. Each number defines the significance on the page.
<p> Paragraph The wrapper elements for all paragraphs.
<a> Link The HTML tag defines hyperlink. It has the following syntax <a href="url" > we can set the target of for tab opening like this em><a href="url" target= "_blank">

CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

Property Name Description
p { color: red; } Color property p is a selector in CSS (it points to the HTML element you want to style). Color is a property, and red is the property value.
body { background-color: lightblue; } Background-color property The background-color property specifies the background color of an element.Similarly we can set any images with different styles for examples if we want to repeat certain image on the page.
body{border-style:} Border property The border-style property specifies what kind of border to display.
p { margin-top: 100px; margin-bottom: 100px; margin-right: 150px; margin-left: 80px; } CSS Margins The CSS margin properties are used to create space around elements, outside of any defined borders. With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).
div { padding-top: 50px; padding-right: 30px; padding-bottom: 50px; padding-left: 80px; } Paddding The CSS padding properties are used to generate space around an element's content, inside of any defined borders. With CSS, you have full control over the padding. There are properties for setting the padding for each side of an element (top, right, bottom, and left).
